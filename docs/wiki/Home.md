# Gait Wiki

This wiki is the practical adoption layer for operators and app teams.

Normative behavior stays in repo docs:

- Primitive contract: `docs/contracts/primitive_contract.md`
- Integration checklist: `docs/integration_checklist.md`
- Mental model: `docs/concepts/mental_model.md`

Use this wiki for playbooks, examples, and onboarding shortcuts.

## Start Here

1. `gait demo --json`
2. `gait verify run_demo --json`
3. Pick your framework recipe in `Integration Cookbook`.
4. Use `Migration Playbooks` to move from demo to production.
